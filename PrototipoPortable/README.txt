Instrucciones de prueba rápida:

1. Extrae PrototipoPortable.zip en otra máquina Windows.
2. Abre la carpeta extraída y ejecuta run.bat.
3. La aplicación prototipo_innovacion.exe intentará copiar la base de datos de recursos y arrancar el backend Node incluido.

Notas:
- No necesitas Rust en la máquina de destino.
- Si el backend no arranca, abre la carpeta Backend y ejecuta node start-packaged.js usando el node.exe dentro de bundled\node.
